# Project Routing

Assessments may route to these destinations, but routing is not adoption.

- **Soho House**: personal/source memory, ledger, project cards, watchlists, and source handles. Store raw/candidate signals here before promotion.
- **Type0**: tenant/newsroom behavior and live editorial system. Shared platform behavior should not be patched only in Type0 when agentic-media/OpenClaw should own it.
- **Gaia**: tenant policy/data/editorial behavior. Prefer shared-platform changes when the same mechanism applies to Type0 and Gaia.
- **agentic-media / OpenClaw**: shared platform/runtime/orchestration/source-authority contracts. Requires stronger tests and adoption-state policy.
- **agentic architecture guide**: general doctrine, not Type0-specific detail. Convert lessons into reusable layer/contract/eval language.
- **agentic-uber-skills**: reusable agentic coding protocols. Add skills only when repeated real use proves extraction makes common paths smaller/faster/safer.

Default decisions:

- Save/watch when relevance is plausible but unproven.
- Convert to eval only when the idea is a failure class worth testing but not yet worth building.
- Reject when source quality, benefit, or fit is weak.
- Adopt now only when evidence, project fit, benefit >> cost, and approval boundary are clear.
