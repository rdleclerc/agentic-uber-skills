# Notice

This repo-local `uber-skill-creator` is a portable skill maintained in `agentic-uber-skills` for Codex, Claude, and SKILL.md-compatible coding agents.

The base scaffold and validation scripts are adapted from OpenAI's Codex `skill-creator` system skill, licensed under Apache License 2.0.

The eval-driven extension adapts public methodology from Anthropic's public skill-creator workflow and community skill-evaluation patterns: intent capture, eval prompts, with-skill vs without-skill comparison, human-reviewable HTML reports, and trigger-description tuning.

No proprietary or leaked Claude Code source is included. Runtime-specific commands and hidden assumptions are intentionally excluded so this skill remains safe for Codex, Claude, and other compatible agent sessions.
